# ビットコインサインツール プロトタイプ開発フロー（簡素化版）

## 現在完了済み ✅
- **基盤レイアウト**: ヘッダー、サイドバー、メイン、フッター
- **ダークテーマ + AIグラデーション**: 完成
- **TradingView Charts基盤**: 初期化・サンプルデータ表示
- **レスポンシブ対応**: 基本対応済み

---

## 残り必要なPhase（4つのみ）

## Phase 1: 時間足切替機能 🎯
**目標**: カスタム時間足ボタン + データ切替
- [ ] 時間足ボタンUI（1M, 5M, 15M, 1H, 4H, 1D）
- [ ] アクティブ状態の視覚化
- [ ] 各時間足の300本データ生成・切替
- [ ] **チェックポイント**: ボタンクリックでチャートデータ切替

## Phase 2: AIサイン表示機能 🎯  
**目標**: Series Markers + サイン追従
- [ ] TradingView Series Markers API実装
- [ ] 買いサイン（青↑）・売りサイン（ピンク↓）表示
- [ ] タイムスタンプベースのサイン管理
- [ ] 時間足変更時のサイン追従機能
- [ ] **チェックポイント**: 時間足切替してもサインが正しい位置に追従

## Phase 3: サイドバー機能 🎯
**目標**: AIサイン履歴 + 統計表示
- [ ] サイン履歴一覧表示
- [ ] パフォーマンス統計（勝率・利益等）
- [ ] サイン詳細情報表示
- [ ] **チェックポイント**: サイドバーに実用的な情報表示

## Phase 4: 最終仕上げ 🎯
**目標**: UX向上 + 多言語対応
- [ ] 多言語切替機能（JP/EN）
- [ ] サイン通知機能
- [ ] アニメーション・UX改善
- [ ] **チェックポイント**: 完成版として満足できるクオリティ

---

## 開発方針 📋
- **段階的開発**: 各Phaseを完全に完了してから次へ
- **チェックポイント重視**: 各Phase後に動作確認必須  
- **モック重視**: 見た目・体験が重要、完璧な機能は不要
- **シンプル実装**: 複雑な実装より確実な動作を優先

## 推定工数
- **Phase 1**: 1-2日（時間足ボタン + データ切替）
- **Phase 2**: 2-3日（サイン表示 + 追従機能）  
- **Phase 3**: 1-2日（サイドバー履歴・統計）
- **Phase 4**: 1日（多言語 + 最終調整）

**合計**: 約1週間でプロトタイプ完成
